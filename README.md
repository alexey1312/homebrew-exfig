# homebrew-exfig

Homebrew tap for [ExFig](https://github.com/alexey1312/ExFig) - fast Figma CLI with parallel exports, batch processing & smart caching for iOS, Android, Flutter, and Web.

## Installation

```bash
brew install alexey1312/exfig/exfig
```

Or tap first, then install:

```bash
brew tap alexey1312/exfig
brew install exfig
```

## Usage

Set your Figma token and start exporting:

```bash
export FIGMA_PERSONAL_TOKEN=your_token_here
exfig --help
```

## Documentation

See the [ExFig documentation](https://alexey1312.github.io/ExFig/documentation/exfig) for full usage instructions.
